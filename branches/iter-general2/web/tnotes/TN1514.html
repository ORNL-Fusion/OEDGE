SOL Option 26. A New Detachment Option Based on DIIID Div Thomson Data.


Dear David and Steve,


I’m attaching Tony Leonard’s 1998 APS poster (which I think you already have) which shows Div Th data for ne and Te in detachment. As you can see the behaviour of Te is about the same as assumed in SOLOpt21, namely pretty well flat and low out to the X-point. In contrast with SOLOpt21, however, the ne and pressure don’t really ramp up all that much through this first region (‘Region A’), but are more like constants. Also, the absolute values for ne in Region A are very high compared to what we usually get with SOLOpt21 - a few 1020 m-3. There is then a pressure rise in a second region (‘Region B’), where ne falls and Te rises. I don’t know how general these results may be, nor how applicable to other tokamaks, but since the DIIID Div Th data are unique, it’s worth basing a new Detached SOLOpt26 on them.


It’s also time to start using outer SOL data to constrain the prescriptions on the inside and to get smooth variations of ne etc at the top.


Region A


Let’s now prescribe distances in terms of the distance from the target to the X-point, which varies from tube to tube, of course. The DIIID data looks like Region A extends to the X-point, but to give ourselves some flexibility, let’s be able to specify the length of this Region as a free parameter times the X-point distance: LA = A*X-pt, where LA [m] is the length of Region A; A is a free specifiable parameter [pure number], default value A = 1; X-pt [m] is the target to X-pt distance for the particular tube.


In this Region we will freely specify both neA and TeA (TiA here), both constants throughout this Region. I realize that will result in conflict with the measured Isat+ values, which we have been thinking should be trustworthy, even in detachment, but before we worry too much about that, let’s just give ourselves this freedom for a start; later we may implement a sub-option where we only specify TeA, and let neA come from the input Isat+. But right now I want to see what the effect is of having such large n­eA values and low TeA values throughout these large volumes, as the Div Th indicates. Later we’ll have to address discrepancies with the Isat+.


One can thus calculate ptotalA in Region A: ptotalA = neAk(TeA + TiA). We’ll not bother about the contribution of dynamic pressure.


For a start, let’s have no prescription for v|| or E||. The idea is to see whether SOLOpt26 is any good for hydrogenic spec signals, where we don’t need to know v|| or E||. Later, if it’s successful and we want to go on to modeling impurities, we’ll have to address this.



Region B


Length of Region B extends from A*X-pt to B*X-pt, where B [pure number] is a second free parameter to specify; default value: B = 1.5, from the Div Th examples.


In this Region Te = Ti again, but T will increase linearly to a specified value at the end of this Region, TB.


Let ptotal ramp up linearly in this Region to the value calculated for the same outside flux tube at the top. This pressure will then be kept constant throughout Region C, thus there will be no break in ptotal at the top.


Knowing the values of T and ptotal at each point, one can calculate ne at each point.


Thus at the end of Region B one knows the values of ne and Te (= Ti).


Region C


Ramp ne, Te, Ti (not equal here) up linearly to the values at the top calculated for the same outside flux tube. Thus there will be no breaks in any of these quantities there.




Thus there will be 5 specifiables: A, B, neA, TA, TB.
